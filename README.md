ZAMBIA WEATHER APP

A program that displays weather readings for zambian cities.

## What it does
- Prints weather readings every 30 seconds
- Sends emails when some conditions exceed or fall below certain levels
  

## Technologies used
- Python
-requests — to fetch weather data from the Open-Meteo API
-schedule — to run the weather check automatically every 30 seconds
-yagmail — to send email alerts when temperature was too high or too low
-time — to keep the scheduler running

## Skills demonstrated
- Automation and scrpting
- API integration
- Scheduling of tasks
- Live readings
